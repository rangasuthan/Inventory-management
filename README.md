# Inventory Management 

A simple **Inventory Management API** built with **FastAPI**, **SQLAlchemy**, and **Pydantic**, using a **MSQL** database.  

```env
# App settings
APP_NAME=Inventory Management API
DEBUG=True
LOG_LEVEL=INFO

# Database settings
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=inventory_db
DB_PORT=3306  # optional, default MySQL port
